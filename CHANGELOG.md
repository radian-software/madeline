## 1.0 (released 2018-08-01)
### Added
* Python package: `madeline`
* Binary: `madeline`
  * Command-line arguments:
    * `--source`
    * `--target`
    * `put | get`
    * `--exclude`
    * `--verbose | --no-verbose`
    * `--identity | --no-identity`
    * `--ssh-add | --no-ssh-add`
  * Environment variable: `MADELINE_DEBUG`
